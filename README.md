# Find Coach

> You can share advert as coach or you can find coach for you and contact them!  

## Technologies

- HTML
- Tailwind 
- Vue.js (Composition API)
- Vue Router
- Pinia 

## Start Setup
With NPM:

```bash
$ npm install mosha-vue-toastify
```

With Yarn:

```bash
$ yarn add mosha-vue-toastify
```
